# Object Segmentation App

Приложение для сегментации и подсчёта объектов на изображениях: перепелиных яиц, красных и жёлтых помидоров черри.

Проект реализован в рамках лабораторной работы по методам сегментации изображений. Основная цель — автоматически находить объекты на изображении, определять их класс и подсчитывать количество объектов каждого типа.

## Возможности

- загрузка изображения через графический интерфейс;
- сегментация объектов на изображении;
- распознавание трёх классов:
  - `quail_egg` — перепелиное яйцо;
  - `yellow_tomato` — жёлтый помидор черри;
  - `red_tomato` — красный помидор черри;
- отображение масок сегментации поверх исходного изображения;
- вывод confidence для каждого найденного объекта;
- подсчёт общего числа объектов и количества объектов каждого класса.

## Используемый подход

В проекте сравнивались классические методы компьютерного зрения и нейросетевой подход.

Классические методы на основе HSV, пороговой обработки и морфологических операций оказались недостаточно устойчивыми: объекты могли склеиваться, яйца плохо отделялись от сложного фона, а красные и жёлтые томаты иногда путались.

Поэтому в качестве основного решения используется сегментационная модель YOLO. Она позволяет не просто находить bounding boxes, а выделять точные маски объектов, что особенно важно при плотном расположении и частичных пересечениях объектов.
